# nuxt.js

> My fabulous Nuxt.js project

## Build Setup

``` bash
# run h2o and node
$ docker-compose up -d

# exec node
$ docker-compose exec node sh

# install dependencies
$ yarn install

# run dev server localhost:80
$ yarn run dev
```

For detailed explanation on how things work, checkout [Nuxt.js docs](https://nuxtjs.org).
